# Simple Calculator App

## Description
This is a basic Python program that takes two numbers as input and prints their sum.

## Features
- Accepts user input
- Performs addition
- Displays result

## Requirements
- Python 3 installed

## How to Run
1. Open terminal
2. Navigate to project folder
3. Run:
   python app.py

## Example
Input:
Enter first number: 4
Enter second number: 5

Output:
Sum is: 9
